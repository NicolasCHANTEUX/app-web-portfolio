**Ambition :** Créer mon clone Spotify pour comprendre comment fonctionnent les grandes plateformes.

### Stack technique
- Backend PHP avec routing personnalisé
- Base de données MariaDB optimisée
- Frontend vanilla JS + API REST

### Défis rencontrés
- Gestion du streaming audio (buffer vs download)
- Optimisation SQL (index, EXPLAIN)
- UX responsive sans framework

**Résultat :** Une app fonctionnelle gérant 500+ titres avec playlist et recherche.
